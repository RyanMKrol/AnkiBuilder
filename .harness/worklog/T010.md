# T010: Thin conversational orchestration layer

## Summary

Authored `skills/build/SKILL.md` — a Claude skill that guides users through the complete pipeline for building an Anki deck. The skill coordinates user decisions (template vs. EPUB, voice choice, deck naming) and orchestrates the underlying CLI commands from T009.

## Approach

The skill is a prose walkthrough that:

1. **Entry decision:** asks user to choose between bundled template (`travel-essentials`) or custom EPUB
2. **Corpus assembly:** runs the appropriate `anki-builder assemble` command and presents an audit table for review
3. **Translation:** explains the `claude -p` based translation and shows the audit table post-translation
4. **Audio:** conditional audio generation with voice choice guidance, referencing ElevenLabs voice IDs
5. **Deck build:** runs `anki-builder deck` with optional custom deck name
6. **Import:** guides final manual import into Anki and verification

## Details

- **Thin Claude layer:** the skill resolves ambiguity (which template? which voice?) and presents audit gates; the CLI carries the logic
- **Practical guidance:** includes setup instructions for `.env`, voice ID examples, troubleshooting, and the complete command reference
