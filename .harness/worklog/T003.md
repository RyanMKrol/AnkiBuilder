# T003: Travel starter corpus template(s) + registry/loader

## 2026-07-13

### What was done

Created the travel-essentials template and template registry/loader:

1. **templates/travel-essentials.json** — A curated travel vocabulary with 36+ entries covering:
   - Greetings (hello, goodbye, thank you, sorry)
   - Numbers (one through one hundred)
   - Directions (left, right, straight, where is...?)
   - Food (water, coffee, bread, menu, bill)
   - Transport (bus, taxi, train, airport, ticket)
   - Money (price, expensive, cheap, currency)
   - Time (now, today, tomorrow, yesterday, hour)
   - Emergencies (help, doctor, hospital, police)
   
   The template sets `meta.sourceType` to `"template"` and includes `targetLanguage: "Spanish"`.

2. **src/corpus/templates.js** — Registry and loader module exporting:
   - `listTemplates()` — returns array of available template names
   - `loadTemplate(name)` — loads and validates a template, throwing a clear error for unknown names

3. **test/corpus/templates.test.js** — Test suite asserting:
   - `listTemplates()` includes "travel-essentials"
   - `loadTemplate()` returns valid schema-compliant corpus objects
   - Travel template includes all 8 required categories
   - Unknown template names raise clear errors
   - All bundled templates validate against the corpus schema
