## Overview

Give users a ready-made "3-day survival" travel vocabulary so they can build a deck without
supplying any source of their own. Travel is the first *purpose*; later purposes are just more
templates, so build a small registry rather than a one-off file.

## Do

Add one or more curated, categorized English travel corpora as data files under `templates/`, and a
registry/loader in `src/corpus/` that lists the available templates and loads one as a valid
`corpus.json` (using the `validateCorpus` + schema from task T002).

## Done when

- At least one template exists (e.g. `templates/travel-essentials.json`) with real English entries
  spread across these categories: Greetings, Numbers, Directions, Food, Transport, Money, Time,
  Emergencies. It sets `meta.sourceType` to `"template"`.
- Every bundled template validates against T002's `corpus.json` schema.
- `src/corpus/templates.js` exports `listTemplates()` (names of available templates) and
  `loadTemplate(name)` (returns a schema-valid corpus object); an unknown name raises a clear error.
- `expectsTest`: `test/corpus/templates.test.js` asserts every bundled template validates,
  `listTemplates()` includes the travel template, `loadTemplate` returns a corpus whose item
  categories include the expected travel categories, and an unknown template name errors.
